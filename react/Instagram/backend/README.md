# Instaclone

Instaclone Backend.

```
npm i --save-dev @babel/core
npm i @babel/preset-env --save-dev
npm i @babel/node --save-dev
npm install apollo-server graphql
npm install prisma --save-dev 
```

Prisma
```
pgadmin4
postgresql app
CREATE DATABASE instaclone
npx prisma init
```
